# 💸 FinanceFlow | Professional Budget & Expense Tracker

[![Python Version](https://img.shields.io/badge/python-3.8%2B-blue)](https://www.python.org/)
[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://streamlit.io/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**FinanceFlow** is a sleek, high-performance personal wealth dashboard designed to give you absolute control over your financial life. Built with a focus on premium aesthetics and intuitive user experience, it features dynamic theming, multi-language support, and powerful data visualizations.

---

## ✨ Key Features

- **📊 Dynamic Dashboard**: Real-time overview of your total balance, monthly income/expenses, and savings rate.
- **💸 Transaction Management**: Effortlessly add, track, and categorize your income and expenses.
- **🎯 Intelligent Budgeting**: Set monthly limits for your categories and get visual progress tracking with status alerts.
- **🌐 Global Ready**: Full support for both **English** and **Turkish** languages, switchable at runtime.
- **🌗 Premium Theming**: Seamless transition between meticulously designed **Light** and **Dark** modes.
- **📈 Data Visualization**: Interactive charts powered by **Plotly** for expense distribution and financial trends.
- **⚙️ Data Export**: Download your entire transaction history as a CSV file for external analysis.

## 🚀 Getting Started

### Prerequisites

- Python 3.8 or higher
- Git

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/BoraKol/Budget-Tracker.git
   cd Budget-Tracker
   ```

2. **Create a virtual environment (optional but recommended):**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Launch the application:**
   ```bash
   streamlit run app.py
   ```

## 🛠️ Tech Stack

- **Frontend/App Framework**: [Streamlit](https://streamlit.io/)
